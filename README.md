## En English Version

### Project Overview
An interactive, bilingual flashcard application designed to demonstrate modern React patterns. Users can toggle between English and French decks, navigate through questions, and flip cards to reveal answers with a premium "Glassmorphism" UI.

###  Technical Features
* **State Management:** Utilizes `useState` to track card indices, language selection, and card flip states.
* **Component Architecture:** Clean separation between the logic-heavy `App` container and the UI-focused `Flashcard` component.
* **Lifecycle Control:** Implements `useEffect` to ensure cards reset to the "question" side automatically when navigating or switching languages.
* **Modern UI/UX:** High-end CSS Glassmorphism using `backdrop-filter`, radial gradients, and smooth transitions for an authentic, classy feel.

###  Technologies
* React (Vite)
* JavaScript (ES6+)
* CSS3 (Flexbox, Backdrop-filter)

## Version Française

### Aperçu du Projet
Une application de cartes mémoire (flashcards) interactive et bilingue conçue pour démontrer les modèles React modernes. Les utilisateurs peuvent basculer entre des jeux de cartes en anglais et en français, naviguer à travers les questions et retourner les cartes pour révéler les réponses.

###  Caractéristiques Techniques
* **Gestion de l'État (State):** Utilisation de `useState` pour suivre l'index des cartes, la sélection de la langue et l'état de retournement.
* **Architecture des Composants:** Séparation nette entre le conteneur `App` (logique) et le composant `Flashcard` (UI).
* **Contrôle du Cycle de Vie:** Implémentation de `useEffect` pour garantir que les cartes reviennent automatiquement du côté "question" lors de la navigation.
* **Interface Moderne:** Design "Glassmorphism" utilisant `backdrop-filter`, des dégradés radiaux et des transitions fluides pour un rendu élégant.

###  Technologies
* React (Vite)
* JavaScript (ES6+)
* CSS3 (Flexbox, Backdrop-filter)
